## Setup
+ Install node.js
+ Clone this project
    ```
	git clone git@github.com:maokailu/components.git
	```
+ Install local dependencies
	```
	npm install
    ```
## Startup

### Development mode
    npm run server

## Test

### Mocha + should
    npm run test
